# 1 RLBook-start

## RL

发展: 试错, 最优控制(价值函数+DP, MC), TD

### 基本特征

1. 试错与延迟奖励， 最大化奖励
2. 抽象为MDP过程
3. 探索与利用的均衡
4. 明确考虑了目标导向智能体与不确定环境交互的问题

###  RL要素

策略, 奖励, 价值函数, 环境模型

### 与进化策略的区别

进化算法忽略了所求策略是状态到动作的函数这一事实, 没有考虑单个状态.

进化方法中, 一般固定策略与与对手博弈多次, 用最后的获胜频率估计获胜概率, 没有考虑中间状态; 而RL方法考虑了单个状态.

进化方法和RL都要对策略空间进行搜索.
